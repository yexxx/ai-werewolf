# AI 狼人杀

一款基于 AI 驱动的现代狼人杀游戏，使用 React、Tailwind CSS 和 大语言模型 (LLM) 构建。您可以与朋友一起玩，也可以挑战能够思考、讨论和制定策略的智能 AI 代理。

![主界面](./imgs/main.png)

## 功能特点

- **多语言支持**: 全面支持英文和中文切换。
- **智能 AI 代理**: AI 玩家使用 LLM 模拟人类的推理和社会推导过程。
- **可自定义 AI**: 为每个 AI 玩家配置不同的 API 端点、模型和密钥。
- **上帝视角**: 开启上帝视角可以查看所有玩家的身份、内心想法和私密行动。
- **配置持久化**: 您的游戏设置和 AI 配置将保存在本地。
- **响应式设计**: 精美的深色主题 UI，适配各种屏幕尺寸。

## 如何开始

1. **设置**: 配置 12 名玩家。您可以选择哪些玩家是人类，哪些是 AI。
2. **AI 配置**: 在全局 AI 设置或单个玩家设置中配置您的 LLM 提供商（如 OpenAI、DeepSeek 等）。
3. **身份分配**: 身份将自动洗牌并分配。
4. **游戏流程**:
   - **夜晚阶段**: 狼人讨论并选择目标。特殊身份（预言家、女巫、守卫）执行各自的行动。
   - **白天阶段**: 玩家讨论昨晚发生的事件，并投票放逐疑似狼人的玩家。
5. **胜负条件**:
   - **好人阵营**: 放逐所有狼人即可获胜。
   - **狼人阵营**: 淘汰足够的平民或神职人员即可获胜。

## 技术栈

- **前端**: React 18, TypeScript, Vite
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **动画**: Framer Motion (Motion)
- **AI**: 通过标准的 OpenAI 兼容 API 集成各种 LLM 提供商。

## 开发指南

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
